# Asset Management Application

This is a full-stack application to manage company assets, including devices and employee assignments.

## Features

- Register new employees
- Add new devices to inventory
- Assign devices to employees
- Track device status (available, assigned, maintenance)
- View assignment history for each device
- View assets assigned to a specific employee

## Tech Stack

- **Backend**: Node.js, Express, SQLite
- **Frontend**: React, TypeScript, Vite
